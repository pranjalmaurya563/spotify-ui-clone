# 🎧 Spotify UI Clone

A clean and modern **Spotify web UI clone** built using pure HTML and CSS.
This project focuses on recreating the layout, styling, and user interface of Spotify’s web player.

---

## 🚀 Features

* Sidebar navigation (Home, Search, Library)
* Music player UI (play button, progress bar, controls)
* Dark theme inspired by Spotify
* Clean and structured layout
* Beginner-friendly frontend project

---

## 🛠 Tech Stack

* HTML5
* CSS3

## ⚙️ How to Run Locally

1. Clone the repository:

   ```
   git clone https://github.com/pranjalmaurya563/spotify-ui-clone.git
   ```

2. Open the folder:

   ```
   cd spotify-ui-clone
   ```

3. Run:

   * Open `index.html` in your browser
